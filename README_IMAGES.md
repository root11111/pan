# 图片资源说明

## 图片下载

所有图片需要从原网站 http://www.zht-lab.cn/ 下载。

### 快速下载方式

1. **使用浏览器扩展**（推荐）
   - 安装图片下载扩展（如 Image Downloader）
   - 访问原网站，批量下载图片

2. **手动下载**
   - 访问原网站
   - 右键点击图片 → 另存为
   - 保存到 `uploads` 目录

3. **使用 Python 脚本**
   - 运行 `scripts/download_images.py`
   - 自动下载所有图片

### 图片存储路径

所有图片应保存在：`D:/workspace/pan/uploads/` 目录

### 注意事项

1. 图片文件名必须与数据库中的路径完全一致
2. 可以通过后台管理系统上传图片替换
3. 建议图片大小不超过 2MB
4. 支持格式：jpg, jpeg, png, gif
