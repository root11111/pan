## 图片处理

### 图片下载
1. 查看 `docs/download_images.md` 了解需要下载的图片列表
2. 使用 `scripts/download_images.py` 脚本自动下载
3. 或手动从原网站下载图片

### 图片路径
所有图片保存在：`D:/workspace/pan/uploads/`

### 通过后台管理上传
1. 登录后台管理系统
2. 进入相应的管理页面
3. 使用图片上传功能上传图片
4. 系统会自动保存并更新数据库

## 注意事项

1. **图片文件名**：必须与数据库中的路径完全一致
2. **内容更新**：所有内容都可以通过后台管理系统修改
3. **数据备份**：执行SQL前建议备份现有数据
4. **图片格式**：支持 jpg, jpeg, png, gif 格式
