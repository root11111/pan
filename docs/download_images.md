## 下载方式

### 方式一：手动下载
1. 访问原网站：http://www.zht-lab.cn/
2. 右键点击图片，选择"另存为"
3. 保存到项目的 `uploads` 目录下对应的子目录

### 方式二：使用下载工具
可以使用浏览器扩展或下载工具批量下载图片。

### 方式三：通过后台管理上传
1. 登录后台管理系统
2. 进入相应的管理页面（公司信息、实验室、新闻等）
3. 使用图片上传功能上传图片
4. 系统会自动保存到 `uploads` 目录

## 图片存储路径

所有图片应保存在：`D:/workspace/pan/uploads/` 目录下

目录结构：
```
uploads/
├── 20231130161735.png
├── 20240119150129.png
├── 20240119150054.png
├── 20240119150032.png
├── 20241217133436948_381613.jpg
├── 20241217133436946_528812.jpg
├── ...
└── img/
    └── 20241217105815.png
```

## 注意事项

1. 确保图片文件名与数据库中的路径一致
2. 图片格式支持：jpg, jpeg, png, gif
3. 建议图片大小不超过 2MB
4. 上传后可以通过后台管理系统随时替换图片
