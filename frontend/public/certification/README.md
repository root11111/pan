# 认证图片下载说明

请从参考网站 https://www.ctl-lab.com/index.html 下载以下认证图片，并保存到本目录：

## 需要下载的图片：

1. `china.png` - 中国认证
2. `fcc.png` - 美国FCC认证
3. `emc.png` - 欧盟EMC认证
4. `ce.png` - 欧盟CE认证
5. `ul.png` - 美国UL认证
6. `gs.png` - 德国GS认证
7. `japan.png` - 日本认证

## 下载方法：

1. 打开参考网站 https://www.ctl-lab.com/index.html
2. 找到"全球认证"模块
3. 右键点击每个认证图标，选择"图片另存为"
4. 将图片保存到 `frontend/public/certification/` 目录
5. 确保文件名与上述列表一致

## 图片格式要求：

- 格式：PNG 或 JPG
- 建议尺寸：200x200px 或更大
- 背景：透明背景（PNG）或白色背景
