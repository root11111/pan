# 检测项目图片下载说明

请从参考网站 https://www.ctl-lab.com/index.html 下载以下检测项目图片，并保存到本目录：

## 需要下载的图片：

1. `medical.png` - 医疗器械
2. `lighting.png` - 灯具
3. `appliance.png` - 家用电器
4. `audio.png` - 视听产品
5. `wireless.png` - 无线通讯
6. `laser.png` - 激光产品
7. `machinery.png` - 机械设备
8. `toy.png` - 玩具
9. `industrial.png` - 工业设备
10. `other.png` - 其他

## 下载方法：

1. 打开参考网站 https://www.ctl-lab.com/index.html
2. 找到"检测项目"模块
3. 右键点击每个检测项目图标，选择"图片另存为"
4. 将图片保存到 `frontend/public/testing/` 目录
5. 确保文件名与上述列表一致

## 图片格式要求：

- 格式：PNG 或 JPG
- 建议尺寸：200x200px 或更大
- 背景：透明背景（PNG）或白色背景
